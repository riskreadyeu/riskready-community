## Module Purpose

The Organisation Module is the foundational component of the RiskReady ISMS platform. It captures and manages all organisational context required for ISO 27001:2022 compliance, including:

- **Organisational Context** (Clause 4.1) - Internal and external issues affecting the ISMS
- **Interested Parties** (Clause 4.2) - Stakeholders and their security requirements
- **ISMS Scope** (Clause 4.3) - Boundaries and applicability of the ISMS
- **Leadership & Governance** (Clause 5) - Roles, responsibilities, and committees
- **Operational Context** (Clause 8) - Business processes, dependencies, and suppliers
